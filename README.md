# Vite + Storybook + Lit

## Install
`npm i`

## Dev work
`npm run dev`

## Storybook
`npm run storybook`

## Storybook static preview
`npm run build-storybook`
`npm run serve:sb`

## Test Storybook
`npm run test-storybook`

## Chromatic
If you want to see chromatic snapshots, you can view in: [Chromatic](https://www.chromatic.com/setup?appId=6304c72060a4a68d6939031c)
`npm run chromatic`
